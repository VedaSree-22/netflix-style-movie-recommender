NETFLIX-STYLE MOVIE RECOMMENDATION SYSTEM

A content-based Movie Recommendation System built using Machine Learning, Flask, and TMDB API with a modern Netflix-style UI.


FEATURES:
Content-Based Filtering using TF-IDF & Cosine Similarity
Real-time Movie Posters from TMDB API
Clean Netflix-style dark UI
Top 10 Similar Movie Recommendations
Fully built using Python + Flask


HOW IT WORKS:
Movie metadata is processed from the TMDB 5000 dataset.
Important features like genres, keywords, cast, and director are combined.
TF-IDF Vectorization converts text into numerical form.
Cosine Similarity finds the most similar movies.
Flask backend serves recommendations to the frontend.
Posters are fetched dynamically using TMDB API.
